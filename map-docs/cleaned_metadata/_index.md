# **CPAD Arctic Map Dataset Directory**

This directory lists all the active dataset layers available on the Arctic Map Explorer website. Use these links to retrieve the detailed metadata and attribute schemas for each layer. To tell researchers how to activate a layer on the website, use the navigation paths listed here.
