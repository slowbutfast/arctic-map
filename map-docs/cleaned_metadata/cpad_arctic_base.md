# **CPAD Arctic Base Boundaries**

**Theme:** Socioeconomic
**Subtheme:** Administrative & Geographic Boundaries
**Layer Name in Database:** `cpad_arctic_base`

**How to Enable on Website Map:**
1. In the left sidebar of the website, expand the **Socioeconomic** category.
2. Expand the **Administrative & Geographic Boundaries** subcategory.
3. Check the box next to **CPAD Arctic Base Boundaries** to display it on the map.

**Title:** CPAD Arctic Base Boundaries
**Abstract:** This layer defines the base geographic boundary for the Co-production of Policy-relevant Climate Data (CPAD) Arctic project. It encompasses the geographic scope of all socioeconomic, environmental, and infrastructure datasets displayed in the Arctic Map Explorer.
**Purpose:** To provide a reference boundary defining the Arctic project area for spatial analysis and mapping.
**Keywords:** CPAD, Arctic, project boundary, reference geography
**Temporal Extent:** 
* **Date**: N/A
* **Time Period**: Active project boundary (present)
**Geographic Extent:**
* **Bounding Box**: Circle enclosing the Arctic region above 50°N latitude.
**Spatial Resolution**: N/A
**Coordinate Reference System**: WGS 1984
**Lineage**: Created by the CPAD project consortium to standardize the visualization boundary for the Arctic region.
**Quality Information:**
* **Accuracy**: Positionally accurate reference boundary.
* **Completeness**: 100% complete boundary polygon.
* **Consistency**: Digitally consistent with all overlay layers.
